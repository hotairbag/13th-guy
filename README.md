# Js13kGames 2024

An entry to the [js13kGames 2024 competition](https://2024.js13kgames.com/).

## Running and building

First, install dependencies with

    $ npm install

To run the game in the browser for development:

    $ npm run dev

On desktop, move with WASD or the arrow keys and press Enter to start. On a
touchscreen, use the virtual joystick to move and the GO button to start,
continue, or restart.

Building the zip package:

    $ npm run build
